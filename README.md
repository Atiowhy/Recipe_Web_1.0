## Installation

Install Mama Recipe Web with npm :

- Clone the Repo

```
  git clone https://github.com/Atiowhy/Recipe_Web_1.0.git
```

- Go To Folder Repo

```
  cd Recipe_Web_1.0
```

- Install Module

```
  npm install
```

- Setup .env

```
VITE_BASE_URL =
```

- Type `npm run start` To Start Production
